### Mystock
* * *
一个实时的股票安卓应用程序，目前支持以下功能：  
1.生成数据，抓取A股sina股票数据  
2.实时更新股票数据  
3.添加、删除、置顶自选的股票  
4.点击某只股票可查看股票分时、日K、周K、月K图  
5.买入、卖出等模拟交易操作。
